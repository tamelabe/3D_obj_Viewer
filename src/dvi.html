3DViewer v2.0 Program Documentation

by tamelabe, yonnarge

General Information

The 3DViewer v2.0 program is an application for visualizing wireframe models in a three-dimensional space.

Model Format

The program supports loading wireframe models from .obj files. Only a list of vertices and faces of the model is supported.

Model Manipulation

In the 3DViewer v2.0 program, you can perform the following actions on the model:

  • Model Translation: specifying the distance relative to the X, Y, Z axes.
  • Model Rotation: specifying the rotation angle relative to its X, Y, Z axes.
  • Model Scaling: specifying the SCALE value.

Graphical User Interface

The graphical user interface of the program is implemented based on Qt and includes the following elements:

  • "LOAD FILE" button for selecting a file with the model and a field for displaying the file name.
  • Wireframe model visualization area.
  • Buttons and input fields for model translation.
  • Buttons and input fields for model rotation.
  • Buttons and input fields for model scaling.
  • Information about the loaded model: file name, number of vertices, and edges.

Limitations

When using the 3DViewer v2.0 program, the following limitations should be taken into account:

  • The program should correctly handle and allow viewing models with up to 1,000,000 vertices without freezing.

Additional Settings

The 3DViewer v2.0 program provides the following additional settings:

  • Projection Type: Parallel and Central.
  • Edge Type: Solid and Dotted.
  • Edge Color and Thickness.
  • Vertex Display Mode: None, Circle, Square.
  • Vertex Color and Size.
  • Background Color Selection.
  • Settings are preserved between program restarts.

Image Capture

The 3DViewer v2.0 program allows saving the generated images in bmp and jpeg formats. It also supports capturing "screencasts" of the current user affine transformations of the loaded object into a gif animation (640x480, 10fps, 5s) by pressing a special button.
