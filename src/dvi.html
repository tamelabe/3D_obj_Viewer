3D Viewer Program

by tamelabe, shoredim

The 3D Viewer Program is a graphical application that allows you to load and manipulate wireframe models from OBJ files. It provides various features for translating, rotating, and scaling the loaded model, and presents a user-friendly interface to interact with the 3D model.

Features:

  • Load a wireframe model from an OBJ file (supports vertices and surfaces list only).
  • Translate the model by a given distance along the X, Y, and Z axes.
  • Rotate the model by a given angle around its X, Y, and Z axes.
  • Scale the model by a given value.
  • GUI implementation based on the Qt library for C++.
  • Visualization area for the wireframe model.
  • Controls for translating, rotating, and scaling the model.
  • Displays information about the uploaded model - file name, number of vertices, and edges.

Performance:

The program efficiently handles models with varying levels of complexity, ranging from a few vertices to up to a million vertices, without any freezing or interface inactivity for more than 0.5 seconds.

Implementation Details:

The program follows the Model-View-Controller (MVC) pattern to ensure separation of concerns. The business logic is kept in the model, while the view is responsible for displaying the data to the user. The controllers act as intermediaries between the model and view, ensuring a thin controller architecture.

Design Patterns Used:

The 3D Viewer Program incorporates several design patterns to enhance its structure and maintainability, including Facade, Strategy, and Singleton patterns.

Usage:

To use the 3D Viewer Program, simply load your wireframe model from an OBJ file using the "Load Model" button. Once the model is loaded, you can perform translation, rotation, and scaling operations using the respective controls provided in the interface.
